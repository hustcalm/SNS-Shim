SNS-Shim
========

A simple wrapper to encapsulate the common SNS APIs, Weibo, Renren, Douban, etc.

## OpenAPI
*   [Sina Weibo](http://open.weibo.com/)
*   [RenRen](http://dev.renren.com/)
*   [Tecent Weibo](http://dev.t.qq.com/)
*   [Douban](http://developers.douban.com/)
